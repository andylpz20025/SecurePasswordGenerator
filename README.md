# SecurePasswordGenerator
Ein kryptografisch sicherer Passwort-Generator für Windows. Nutzt den RandomNumberGenerator für hohe Entropie und bietet einen Streaming-Schutz via Windows-API, um Videoaufnahmen (OBS/Discord) zu blockieren. Ermöglicht die Generierung in frei konfigurierbaren Blöcken für maximale Sicherheit und Übersichtlichkeit.
